# X Live

X Live supports real-time broadcasts for product launches, conferences, watch parties, interviews, entertainment, sports, and other live moments.

## Best Uses

- Product announcements.
- Live events and conferences.
- Sports or entertainment companion content.
- Executive/founder Q&A.
- Creator or publisher partnerships.

## Planning Inputs

| Input | Why it matters |
|---|---|
| Event date and run-of-show | Live campaigns need precise timing |
| Hosts and speakers | Drives credibility and attendance |
| Promotion window | Build reminders before the live moment |
| Post-live plan | Clips and retargeting can extend value |
| Moderation | Live conversation needs operational coverage |

## Campaign Structure

```text
Pre-event: awareness, reminders, Branded Notifications where available
Live: X Live promotion and real-time engagement
Post-event: highlight clips, retargeting, conversion follow-up
```

## Creative Guidance

- Make the topic and time clear.
- Use short teaser clips.
- Promote why the audience should attend live.
- Prepare clips for post-event amplification.

## Measurement

| Stage | Metrics |
|---|---|
| Pre-event | Reach, reminder opt-ins, site registrations |
| Live | Viewers, watch time, engagement, conversation volume |
| Post-event | Clip views, traffic, leads, conversions |

## Common Mistakes

- Promoting only on the day of the event.
- No clear reason to watch live.
- No post-event content plan.
- No moderation or response plan.
