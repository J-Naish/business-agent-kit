# Image Ads

Image Ads are Promoted Ads that combine one image with post copy and optional X Ad Features such as Website Cards, App Cards, Conversation Buttons, or Polls.

## Specs

| Item | Guidance |
|---|---|
| Recommended formats | PNG or JPEG |
| File size | Max 5 MB |
| Common aspect ratios | 1:1, 1.91:1, 4:5, 2:3, 16:9, 9:16 where supported |
| Common sizes | 1080x1080, 2064x1080, 1440x1800, 1080x1620, 1920x1080, 1080x1920 |
| Copy | Up to 280 characters; links reduce available copy length |

X expanded aspect-ratio support in 2026. Verify the selected placement and creative preview before launch.

## Feature Variants

| Variant | Use |
|---|---|
| Standalone Image Ad | Brand message, product proof, announcement |
| Website Card | Drive traffic to a landing page |
| App Card | Drive app installs or app opens |
| Conversation Button | Encourage users to post with prefilled text or a hashtag |
| Poll | Create lightweight interaction and audience feedback |

## Creative Practices

- Make one focal point obvious.
- Keep text overlays minimal and readable on mobile.
- Use high-contrast imagery that stands out in the timeline.
- Show the product, outcome, proof, or use case quickly.
- Keep important content away from edges because cropping can vary.
- Match the image, copy, card headline, and landing page.

## Copy Practices

Use copy to explain why the visual matters.

```text
Hook: why the user should care
Benefit/proof: what changes or why to believe
CTA: what to do next
```

## Funnel Use

| Funnel stage | Role |
|---|---|
| Awareness | Brand/product introduction |
| Consideration | Proof, comparison, feature education |
| Conversion | Offer, product detail, landing page click |
| Retargeting | Reminder, objection handling, incentive |

## Common Mistakes

- Too much text in the image.
- Multiple products or messages competing for attention.
- Generic stock photography.
- Weak message match between ad and landing page.
- Low-resolution or heavily compressed assets.
