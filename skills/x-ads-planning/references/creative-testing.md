# Creative Testing

Use this reference when planning X Ads creative tests, refresh cadence, or message experiments.

## Testing Principles

- Test one major learning at a time: message, audience thesis, format, offer, visual, or destination.
- Use X-native post craft before testing micro-variants.
- Keep ad groups readable. If many audiences and creatives are mixed together, the result cannot explain what worked.
- Use replies, not just metrics, as creative research.
- Do not pause or rebuild because of one noisy day.

## Post Count

X official optimization guidance recommends multiple posts per campaign and says higher-ranked posts serve more often. It recommends at least **3 posts per campaign** and no more than **7-10** in one campaign.

Practical posture:

| Situation | Creative count |
|---|---|
| Small test | 3 materially different posts |
| Normal launch | 3-5 posts with distinct hooks/angles |
| Larger test | 7-10 only when budget and audience size can give each a read |
| Retargeting | Fewer posts, refreshed more often; watch frequency and replies |

Avoid stuffing campaigns with near-identical posts.

## Test Types

| Test | Good design | Bad design |
|---|---|---|
| Message angle | Same audience, different claim/proof | Different audience, bid, creative, and landing page |
| Hook line | Same offer/visual, different first line | Minor punctuation changes |
| Format | Text vs image vs video vs vertical video | Cross-format test with different offer and audience |
| Audience thesis | Keyword vs follower look-alike vs conversation | Layering all targeting types together |
| Creator/handle | Founder/creator/brand handle where workflow allows | Assuming one identity always wins |
| Destination | Same post to page vs card/destination variant | Changing targeting and creative too |

## Refresh Signals

Use account-local baselines. Practical triggers to investigate:

- CTR drops materially versus the post's early stable period.
- CPA/CPL/CPI rises while landing page or app conversion rate is unchanged.
- Reply sentiment shifts negative or repetitive.
- Frequency rises in a small retargeting or Custom Audience pool.
- One post receives most impressions and the rest never get a meaningful read.
- CPM jumps without a budget or market explanation; audit creative quality, brand safety, and Aesthetic Score guidance.

## Concept vs Variant

| Level | Example |
|---|---|
| Concept | Founder POV vs customer proof vs competitor alternative |
| Hook | First line or first frame |
| Proof | Quote, stat, demo, screenshot, review, case result |
| Variant | Crop, caption, CTA, end card, minor copy polish |

On X, copy variants can matter more than on more visual platforms, but they still need a real message difference.

## Decision Rules

- Use 3-7 day rolling reads for conversion campaigns unless delivery is broken.
- Fix tracking, disapproval, or destination problems immediately.
- Do not judge a keyword campaign before it has enough delivery and search/keyword data.
- If CTR is weak, diagnose first line, visual, and audience context.
- If CTR is healthy but CVR is weak, diagnose destination, offer, event, and lead/order quality.
- If engagement is strong but business outcome is weak, the ad may be starting conversation without creating demand.
