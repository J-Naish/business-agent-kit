# Carousel Ads

Carousel Ads are Promoted Ads with multiple horizontally swipeable images or videos. Use them to show multiple products, explain a sequence, or build a narrative.

## Specs

| Item | Guidance |
|---|---|
| Slides | 2-6 images or videos |
| Image carousel sizes | Common specs include 800x418, 800x800, and expanded ratios such as 1440x1800, 1080x1620, 1080x1080, 2064x1080, 1920x1080, 1080x1920 |
| Video carousel sizes | Common specs include 800x450, 800x800, and expanded ratios where supported |
| Aspect ratio | Keep one aspect ratio within a carousel |
| Mixed media | Keep image and video aspect ratios consistent; 1:1 is the safest mixed-media choice |
| Destination | One destination for single-destination carousels; up to six unique web destinations for multi-destination Website Carousels |
| Website title | Up to 70 characters; 50 or fewer is safer for truncation |

X expanded 4:5 and 2:3 support in 2026 to improve creative portability from Meta, TikTok, Instagram, and other social platforms. Verify current creative specifications in X Ads Manager before launch.

## Best Uses

- Multiple products or features.
- Step-by-step explanations.
- Before/after sequences.
- Benefit stack.
- App or SaaS onboarding flow.
- Event agenda or speaker lineup.

## Story Structures

| Pattern | Cards |
|---|---|
| Product set | Product A -> Product B -> Product C -> CTA |
| Story | Problem -> insight -> solution -> proof -> CTA |
| Feature sequence | Feature 1 -> Feature 2 -> Feature 3 -> value -> CTA |
| Objection handling | Objection -> answer -> proof -> next step |
| Before/after | Before -> process -> after -> offer |

## Creative Practices

- Make card 1 a strong hook.
- Keep card design consistent.
- Use short, readable text.
- Keep each card focused on one idea.
- Use visual continuity to encourage swiping.
- Match each destination to the card promise.

## Common Mistakes

- Treating the carousel as a random image dump.
- Mixing unrelated aspect ratios.
- Using tiny text.
- Repeating the same idea across all cards.
- Sending all cards to a generic landing page when specific destinations would be better.
