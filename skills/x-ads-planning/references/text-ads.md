# Text Ads

Text Ads are Promoted Ads that look and behave like standard posts. Use them when the message itself is the creative.

Spec source: X Creative Ad Specifications (https://business.x.com/en/help/campaign-setup/creative-ad-specifications). Last verified: 2026-04.

## Specs

| Item | Guidance |
|---|---|
| Post copy | Up to 280 characters; links reduce available copy length |
| Media | None required |
| Engagement | Can receive replies, reposts, likes, and other standard X interactions |
| Best objectives | Reach, Engagement, Website traffic, Website conversions / Sales when the message and destination are strong |

Verify current character handling and link behavior in X Ads Manager before launch.

## Best Uses

- Announcements and launches.
- Founder or brand point of view.
- Fast message testing before producing visual assets.
- Promoting an organic post that already has strong engagement.
- Thread or conversation entry points.
- B2B/SaaS offers where the idea, POV, or proof is stronger than the visual.

## Creative Practices

- Lead with a clear point of view or benefit.
- Put the hook in the first line; do not bury the main point below truncation.
- Keep one message per ad.
- Avoid generic corporate language.
- Avoid raw URLs when a Website Card is a better click experience.
- Avoid hashtags in ordinary promoted ad copy unless the selected X-managed feature explicitly requires one.

## Copy Structure

```text
Opening line: sharp hook or audience callout
Middle: why it matters or proof
Close: clear next action
```

## Common Mistakes

| Mistake | Why it hurts |
|---|---|
| Writing like a banner headline | Text Ads should feel native to X |
| Too many claims | The post becomes hard to understand |
| No action path | Engagement may not translate into business outcomes |
| Hashtag stuffing | Looks low quality, can dilute the message, and may conflict with current paid-copy restrictions |

## Measurement

Judge Text Ads by the job:

- Awareness: reach, frequency, CPM.
- Engagement: engagement rate, CPE, conversation quality.
- Traffic/conversion: CPC, landing page behavior, conversion rate, CPA.

## Diagnosis

| Symptom | Likely cause | Response |
|---|---|---|
| Low engagement | Message feels generic or ad-like | Rewrite with a sharper point of view or audience callout |
| Engagement but no action | No clear path or CTA | Add destination logic or pair with a card/media format |
| Negative replies | Claim, tone, or audience mismatch | Review policy, comments, and brand-safety posture |

## Operating Cadence

- Use Text Ads for fast message testing before producing heavier creative.
- Promote organic posts only when the organic engagement is relevant and high quality.
- Retire messages that drive low-quality replies even if engagement volume is high.
