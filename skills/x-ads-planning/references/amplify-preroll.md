# Amplify Pre-roll

Amplify Pre-roll lets advertisers run pre-roll video ads against premium video content categories on X.

Spec source: X Creative Ad Specifications (https://business.x.com/en/help/campaign-setup/creative-ad-specifications). Last verified: 2026-04.

## Best Uses

- Brand awareness.
- Product launches.
- Event promotion.
- Entertainment, sports, news, and culture-adjacent campaigns.
- Campaigns that need brand-safe video context.

## Buying and Eligibility

Amplify Pre-roll can be available through self-serve or eligible account workflows, depending on market, account, and inventory. Verify current availability in X Ads Manager or with X support.

## Creative Guidance

| Element | Guidance |
|---|---|
| File type | MP4 or MOV |
| File size | 1 GB max |
| Length | 15 seconds or less recommended; 2:20 max |
| Aspect ratio | 1:1 recommended; 9:16 can take similar real estate; mirror publisher content aspect ratio when possible |
| Size | 1200x1200 recommended; 600x600 minimum for 1:1; 640x360 minimum if not 1:1 |
| Opening | Brand or product cue early |
| Message | One clear idea |
| Sound | Design for sound-on but include captions or text overlays |
| CTA | Use when the campaign has a downstream action |

## Targeting and Context

- Select content categories that fit the brand and message.
- Avoid category choices that create brand safety or contextual mismatch.
- Align creative tone with the premium video environment.

## Measurement

| Goal | Metrics |
|---|---|
| Awareness | Reach, CPM, frequency |
| Video consumption | Views, view rate, CPV, completion rate |
| Downstream action | Site visits, conversions, assisted impact |

## Common Mistakes

- Using a slow TV-style opening.
- Treating pre-roll as a direct-response-only format.
- Ignoring contextual fit.
- Judging only by last-click conversions.

## Diagnosis

| Symptom | Likely cause | Response |
|---|---|---|
| Low completion | Creative too slow or too long | Cut to the brand/product cue and core message earlier |
| Weak brand effect | Category/context mismatch | Revisit content categories and moment fit |
| Poor downstream action | No post-view plan | Pair with retargeting or follow-up Promoted Ads |

## Operating Cadence

- Confirm inventory/category availability before building the plan.
- Review view quality and brand safety weekly.
- Use downstream reads as supporting evidence, not the only success metric.
