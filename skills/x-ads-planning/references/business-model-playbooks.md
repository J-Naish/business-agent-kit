# Business Model Playbooks

Use this reference before choosing X objectives or ad formats. Start from the business model, then choose the campaign menu items that support it.

## E-commerce / Retail

Default strategy:

- Use **Website conversions / Sales** when purchase tracking is ready.
- Use **Dynamic Product Ads** only when X Shopping Manager, feed quality, and product events are ready.
- Separate prospecting and retargeting when budget, reporting, or audience intent differs.
- Track purchase value, contribution margin, new vs returning customers, and product-level performance.

Recommended structure:

| Role | Objective / format | Notes |
|---|---|---|
| Prospecting | Website conversions / Sales with Image, Video, Carousel, or DPA Prospecting | Use broad enough audiences and strong product/category hooks |
| Retargeting | DPA Retargeting or product-focused Carousel | Use catalog activity and site audiences where available |
| Creative testing | Image/Video/Carousel | Test offers, product categories, proof, and problem framing |

Avoid judging only on platform ROAS. X can over-credit branded or retargeting demand, especially when other paid social/search channels are active.

## Lead Generation

X does not have a current native lead form objective equivalent to Meta Lead Ads or LinkedIn Lead Gen Forms. Treat lead generation as a website conversion problem.

Default strategy:

- Use **Website conversions / Sales** if form-fill, qualified lead, or CRM-stage events are implemented.
- Use **Website traffic** only when conversion tracking is not ready or the goal is content consumption.
- Use **Keyword Ads**, follower look-alikes, conversation targeting, and retargeting to shape intent.
- Import or reconcile qualified lead quality outside X Ads Manager whenever possible.

Recommended structure:

| Role | Objective / format | Notes |
|---|---|---|
| Intent capture | Keywords campaign / Search Ads | Match copy to search or problem intent |
| Prospecting | Image/Video/Carousel to landing page | Use proof, pain, outcome, and objection handling |
| Retargeting | Image/Video/Carousel | Address objections and push to conversion |

Watch lead quality, not only CPL. Cheap leads can train delivery toward low-intent form fills.

## SaaS / B2B

Default strategy:

- Use X for category conversations, competitor/alternative discovery, founder/thought-leadership amplification, and retargeting.
- Use **Website conversions / Sales** for demo, trial, signup, or qualified signup events when volume supports it.
- Use **Website traffic** for content offers and education only when downstream tracking is clear.
- Use Keyword Ads and follower look-alikes carefully; match each ad group to a clear audience hypothesis.

Recommended structure:

| Role | Objective / format | Notes |
|---|---|---|
| Category intent | Keywords campaign / Search Ads | Problem, category, competitor, and comparison terms |
| Education | Video, Image, Carousel | Show problem, proof, workflow, or differentiated POV |
| Retargeting | Image/Video/Carousel | Move site visitors or content engagers toward trial/demo |

Track pipeline quality and activation, not only trial/signup volume.

## App

Default strategy:

- Configure an approved Mobile Measurement Partner before planning app-install bidding.
- Use **App installs** for acquisition and **App re-engagements** for existing users.
- Separate iOS/Android or major geos only when budget and measurement justify it.
- Optimize beyond installs as soon as meaningful in-app event volume is available.

Recommended structure:

| Role | Objective / format | Notes |
|---|---|---|
| Acquisition | App installs with Video, Vertical Video, Carousel, App Card | Demonstrate the app in the first seconds |
| Re-engagement | App re-engagements with deep links | Use feature, offer, or lifecycle-specific messages |
| Brand halo | Reach or Video views with app conversion measurement enabled | Use only when upper-funnel value is plausible |

## Brand / Launch

Default strategy:

- Use **Reach** and **Video views** for broad exposure.
- Use Vertical Video Ads and Video Ads as the core creative formats.
- Consider Timeline Takeover, Spotlight Takeover, Amplify, Branded Hashtags, or Branded Notifications only when the campaign has enough budget, timing, and operational support.

Recommended structure:

| Role | Objective / format | Notes |
|---|---|---|
| Reach burst | Reach + Vertical Video / Takeover | Use for launches and cultural moments |
| Storytelling | Video views + Video/Vertical Video | Optimize for view quality, not only cheap views |
| Conversation | Engagement + branded features where eligible | Requires community management and moderation |

Use lift, search/direct traffic, branded search, site visits, and qualitative conversation review where possible.

## Live / Event

Default strategy:

- Plan in three phases: pre-event, live, post-event.
- Use Branded Notifications only when managed access, Arrow setup, and a real reminder value exist.
- Use X Live and clips when the event has a reason to be watched live.

Recommended structure:

| Phase | Formats | Measurement |
|---|---|---|
| Pre-event | Video, Image, Branded Notifications, Takeover | Reach, reminders, registrations |
| Live | X Live, Promoted clips, engagement ads | Viewers, watch time, engagement |
| Post-event | Highlight clips, retargeting, conversion ads | Clip views, site actions, leads, sales |

## Multi-format Patterns

Use multiple formats when each has a distinct job:

| Pattern | Use when | Mix |
|---|---|---|
| Performance launch | Conversion tracking is ready | Website conversions / Sales + Image/Video + retargeting |
| Catalog commerce | Feed and events are ready | DPA Prospecting + DPA Retargeting + Collection |
| Intent plus education | Search intent exists but needs explanation | Keywords campaign + Video/Carousel + retargeting |
| Cultural moment | Timing and conversation context matter | Vertical Video + Takeover/Amplify + engagement follow-up |
| App growth | MMP and in-app events are ready | App installs + Vertical Video + re-engagement |
