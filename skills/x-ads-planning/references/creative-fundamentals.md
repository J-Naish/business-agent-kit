# Creative Fundamentals

Use this reference before writing X Ads copy or specifying creative assets.

## Core Principles

- Make the first line or first frame earn attention quickly.
- Use one ad for one message.
- Match the creative to the destination, card, or app action.
- Use native X behavior: concise language, timely relevance, conversation context, and clear point of view.
- Keep hashtags, emojis, raw URLs, and clutter minimal unless the format explicitly needs them.
- Design mobile-first and verify creative previews in X Ads Manager.

## Aesthetic Score Guidance

X's Aesthetic Score is a creative-quality signal. Treat it as a quality filter, not a substitute for business outcomes.

Common practices:

- Avoid hashtags in regular ad copy unless using a Branded Hashtag or Conversation Button strategy.
- Avoid raw URLs in copy; use Website Cards or App Cards when a click destination is needed.
- Keep copy grammatically clean and easy to scan.
- Use polished, high-resolution assets.
- Avoid low-quality screenshots, cluttered text overlays, and excessive promotional language.
- Refresh creative when fatigue appears.

## Copy Framework

```text
Hook: earn attention in the first line
Value: explain the benefit, proof, or reason to care
Action: tell the user what to do next
```

## Hook Types

| Hook type | Use |
|---|---|
| Problem | Name the pain or friction |
| Outcome | Lead with the desired result |
| Proof | Use data, review, credibility, or social proof |
| Contrast | Old way vs new way, before vs after, competitor alternative |
| Timely angle | Tie into a current event, season, launch, or conversation |
| Direct callout | Name the audience or situation clearly |
| Question | Prompt self-identification or curiosity |

## CTA Selection

| CTA | Best for |
|---|---|
| Learn More | Education, consideration, B2B, services |
| Shop Now | E-commerce and product sales |
| Sign Up | SaaS, webinars, newsletters, trials |
| Download | Apps, reports, guides |
| Install | App installs |
| Watch | Video or event content |
| Apply Now | Applications, hiring, finance, education |

## Format Fit

| Format | Creative job |
|---|---|
| Text Ads | Native message, announcement, opinion, fast test |
| Image Ads | One product, offer, visual proof, or brand message |
| Video Ads | Demo, story, testimonial, proof, education |
| Vertical Video Ads | Full-screen mobile story, fast hook, sound-on impact |
| Carousel Ads | Multi-step story, product set, benefit stack |
| Collection Ads | Product discovery and multi-destination shopping |
| Dynamic Product Ads | Catalog-personalized retargeting or prospecting |

## Testing

Test concepts before cosmetic variants.

| Test level | Examples |
|---|---|
| Offer | Discount, trial, bundle, lead magnet, event registration |
| Angle | Pain, outcome, proof, comparison, urgency |
| Format | Text, image, video, carousel, vertical video |
| Hook | First sentence or first 1-3 seconds |
| Proof | Testimonial, rating, creator, data, demo |
| Variant | Crop, headline, CTA, color, opening frame |

Use several meaningfully different creatives when budget supports it. Avoid launching many near-identical variants that split learning without answering a strategic question.
