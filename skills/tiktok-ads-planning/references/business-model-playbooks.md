# Business Model Playbooks

Use this reference before selecting formats. It keeps TikTok planning tied to business economics rather than ad-format menus.

## Website E-commerce / DTC

| Layer | Default recommendation |
|---|---|
| First campaign | Sales / website conversion with In-Feed or Spark; Smart+ Web when signal and creative supply are ready |
| Expansion | Search Ads for active product/category intent; Smart+ Catalog Ads if catalog is healthy |
| Creative | Demo-first, review/comment proof, founder/creator POV, comparison, offer clarity |
| Measurement | Pixel + Events API, `event_id` deduplication, Shopify/commerce source of truth, margin |
| Watch-out | TikTok may create demand that converts via Google, Amazon, direct, or later retargeting |

Design notes:

- Do not judge only on platform ROAS.
- Keep landing pages mobile-fast and matched to the ad promise.
- Use Search Ads to capture "best X", review, price, and problem/solution queries.
- Treat cheap clicks as suspect until order quality is proven.

## TikTok Shop / Live Commerce

| Layer | Default recommendation |
|---|---|
| First campaign | Product GMV Max for Shop sales; LIVE GMV Max only when LIVE operations are real |
| Creative | Creator/affiliate videos, product demos, Spark where eligible, Shop-native proof |
| Operations | Listings, price, reviews, inventory, fulfillment, returns, affiliate program |
| Measurement | Seller Center, net revenue, returns, contribution margin, new buyer rate |
| Watch-out | GMV is not profit; automation still needs product and creative governance |

Design notes:

- Product readiness is media readiness.
- Exclude products with poor margin, return risk, or fulfillment issues.
- Build creator-affiliate supply; GMV Max cannot optimize assets it does not have.
- LIVE commerce needs host, run-of-show, inventory, offers, and enough session duration.

## Lead Generation

| Layer | Default recommendation |
|---|---|
| First campaign | Lead Generation with Instant Form; website form when compliance/education needs are complex |
| Form posture | More Volume for simple offers; Higher Intent for quality-sensitive offers |
| Creative | Self-qualifying hooks, proof, examples, clear follow-up promise |
| Measurement | CPL, qualified lead rate, contact rate, appointment/sales acceptance, revenue |
| Watch-out | Cheap CPL can be a failure if close rate is low |

Design notes:

- Define qualified lead before launch.
- Connect CRM/Leads Center before scaling.
- Use speed-to-lead SLA.
- Daypart when response teams are not staffed.

## Apps and Games

| Layer | Default recommendation |
|---|---|
| First campaign | App Promotion with install or in-app event optimization |
| Expansion | Smart+ App, Playable Ads, Pangle where quality supports it |
| Creative | Gameplay/value demo, problem solved in app, creator walkthrough, app-store match |
| Measurement | MMP or TikTok SDK, SKAN/iOS settings, in-app event, retention, LTV |
| Watch-out | Cheap installs can hide poor retention |

Design notes:

- Optimize beyond install when event volume supports it.
- Diagnose Pangle separately from TikTok feed.
- Use playable only when the demo honestly represents app value.

## B2B / SaaS

| Layer | Default recommendation |
|---|---|
| First campaign | Lead Generation or website conversion; Search Ads for problem/category intent |
| Creative | Founder/expert POV, pain-point education, objection handling, demo snippets |
| Destination | Guide, webinar, trial, consultation, or demo flow with clear qualification |
| Measurement | Qualified lead, sales accepted lead, pipeline, CAC/payback |
| Watch-out | TikTok can create demand but may not close long-cycle intent directly |

Design notes:

- Avoid treating all form fills as equal.
- Use content that teaches and qualifies.
- Retarget video viewers and site visitors with proof or next-step offers.

## Local / Service Businesses

| Layer | Default recommendation |
|---|---|
| First campaign | Lead Generation, Messaging Ads, or website conversion depending on response flow |
| Creative | Local proof, before/after, staff/owner POV, service process |
| Targeting | Geo control matters; broad national guidance may not apply |
| Measurement | Calls/messages/bookings/qualified appointments/revenue |
| Watch-out | Response speed and scheduling capacity determine profitability |

Design notes:

- Use messaging only when the inbox is staffed.
- Use Higher Intent when lead quality matters more than CPL.
- Keep claims and regulated-category requirements tight.

## Brand Awareness / Launches

| Layer | Default recommendation |
|---|---|
| First campaign | Reach & Frequency, TopView/Top Feed/TopReach where managed access exists, In-Feed/Spark support |
| Creative | One sharp message, native pacing, creator seeding, proof or participation mechanic |
| Measurement | Reach, frequency, view quality, brand lift, search/direct lift, retargeting pool growth |
| Watch-out | Last-click CPA is usually the wrong primary KPI |

Design notes:

- Pair premium reach with auction follow-up.
- Use Branded Mission/Effects/Hashtag only when participation has a real behavior to copy.
- Define proxy demand signals before launch.
