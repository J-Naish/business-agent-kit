# Search Ads

Search Ads help advertisers reach people when they search on TikTok. Use them when keyword intent, product research, problem-solving, or category discovery matters.

## Best Uses

- High-intent product/category searches.
- Problem/solution queries.
- Competitor or alternative research where policy allows.
- App discovery.
- Educational content that answers a search question.
- Retargeting users who have shown search or content intent.

## Keyword Strategy

Current official Search Ads Campaign guidance supports Web Conversion and Traffic objectives, Landing Page View optimization, keyword-level bidding within the Traffic objective, and keyword/negative-keyword controls where available. Availability varies by market and account.

| Keyword type | Use |
|---|---|
| Brand | Defend or amplify brand demand |
| Category | Capture shoppers/researchers |
| Problem | Match pain-point searches |
| Competitor/alternative | Use carefully and follow policy |
| How-to | Education and product demonstration |
| Event/seasonal | Timely demand capture |

### Match Types

| Match type | Planning use |
|---|---|
| Broad match | Highest reach, lowest query precision |
| Phrase match | Balance reach and relevance around the phrase meaning |
| Exact match | Highest relevance, lowest reach; close variants can still apply |
| Negative keywords | Block irrelevant or unsafe query themes; use broad, phrase, or exact negative match where available |

Launch with tightly themed ad groups. TikTok recommends at least 20 keywords per Search Ads Campaign ad group in current guidance.

## Creative Guidance

- Answer the search intent in the first seconds.
- Put the keyword concept or solution on screen early.
- Use demo, proof, or comparison rather than generic brand awareness.
- Align the landing page with the keyword group.
- Use Spark Ads only when the post clearly matches the search intent.

## Campaign Notes

- Verify current Search Ads availability, market support, match types, keyword limits, and negative keyword options in TikTok Ads Manager.
- Keep ad groups thematically tight.
- Use UTMs and landing-page analytics to evaluate search traffic quality.
- Start with broad match where appropriate to give the system reach and learning, then move proven terms to phrase/exact if tighter control is needed.
- Use the Search Terms Report as both a negative-keyword source and a keyword expansion source.
- Avoid pausing/relaunching when keyword, negative, bid, or creative optimization can happen in the existing campaign.

## Query-to-Creative Mapping

| Query type | Creative job |
|---|---|
| Best / top / review | Comparison and proof |
| How to / fix / solve | Education and demo |
| Brand query | Official source, offer, product range, trust |
| Competitor / alternative | Policy-safe comparison and differentiator |
| Price / discount | Value stack and offer clarity |
| Problem | Pain-point hook and solution proof |

If the creative does not answer the query in the first seconds, the keyword is probably too broad for that asset.

## Common Mistakes

- Using broad, generic videos for high-intent keywords.
- Mixing very different intent levels in one ad group.
- Judging only by CTR instead of conversion quality.
- Forgetting negative keywords where available.
- Assuming negative keywords block plurals, misspellings, and synonyms automatically. Add variants deliberately.
