# Carousel Ads

Carousel Ads use multiple images in a swipeable format. Use them when still images can explain the product, sequence, or offer better than a single video.

## Best Uses

- Product sets and category browsing.
- Step-by-step explanation.
- Before/after or transformation stories.
- Feature comparison.
- App or SaaS walkthrough.
- Retargeting with product or proof sequences.

## Creative Guidance

| Element | Guidance |
|---|---|
| First card | Must work as the hook |
| Story order | Each card should advance the idea |
| Visual system | Keep aspect ratio, type, and design consistent |
| Text | Keep on-image text short and readable |
| CTA | Make the final action obvious |
| Music | Standard Carousel Ads require music in current specs; use CML or eligible uploaded audio as allowed |

## Story Patterns

| Pattern | Card flow |
|---|---|
| Product set | Hero product -> variants -> benefits -> offer |
| Problem/solution | Pain -> why it happens -> solution -> proof -> CTA |
| Tutorial | Step 1 -> step 2 -> step 3 -> result |
| Comparison | Old way -> new way -> proof -> next step |
| Social proof | Review -> use case -> result -> CTA |

## Campaign Notes

- Use Traffic, Sales/Web Conversion, App Promotion, or Reach where available.
- Verify card count, file size, aspect ratio, caption, CTA, and music limits in TikTok Ads Manager. Current specs distinguish Standard Carousel and VSA Carousel requirements.
- Use separate destinations per card only where the account/format supports it.
- Avoid treating "video carousel" as a separate official product unless the current account UI names it that way.

## Common Mistakes

- Treating the carousel as a random image dump.
- Putting too much text on each card.
- Using inconsistent crops or design.
- Sending all cards to a generic destination when specific destinations would convert better.
