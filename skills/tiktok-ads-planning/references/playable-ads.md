# Playable Ads

Playable Ads are interactive ad experiences, often used for apps and games, that let users try a simplified experience before installing or converting.

Playable Ads are often used with Pangle. Pangle is TikTok for Business's external app network, not TikTok in-feed inventory, so diagnose it separately from TikTok placements.

## Best Uses

- Mobile games.
- Apps with a clear interactive demo.
- Utility apps where the core value can be simulated.
- Pangle or app inventory where playable inventory is eligible.

## Creative Requirements

Playable Ads usually require an HTML5 package or supported playable asset. Verify current TikTok/Pangle requirements before production.

| Element | Guidance |
|---|---|
| First interaction | Make it obvious within seconds |
| Demo length | Keep the experience short and rewarding |
| CTA | Install or continue action should appear naturally |
| File/package | Follow current upload and QA requirements |
| Performance | Must load quickly and work across devices |

## Measurement

Track:

- Interaction start rate.
- Completion rate.
- Click/install rate.
- CPI and post-install event quality.
- Creative-to-app-store message match.

## Common Mistakes

- Making the playable too complex.
- Hiding the value until late.
- Misrepresenting actual app/gameplay.
- Ignoring load time and device compatibility.
