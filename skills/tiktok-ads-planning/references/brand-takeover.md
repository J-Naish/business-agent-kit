# Brand Takeover

Brand Takeover is a premium managed format for short, high-impact exposure where available. Treat it as a reservation product that requires TikTok confirmation for market, eligibility, specs, and pricing.

Use conservative language: Brand Takeover is still referenced, but current premium planning often centers on TopView, TopReach, and Logo Takeover. Do not make Brand Takeover the default app-open recommendation without current TikTok confirmation.

## Best Uses

- Large launches.
- High-reach brand moments.
- Category or seasonal bursts.
- Campaigns where exclusivity or first exposure matters.

## Creative Guidance

- Keep the message extremely simple.
- Use one product, offer, or launch cue.
- Make the brand visible immediately.
- Design for very short attention windows.
- Use follow-up formats for explanation.

## Planning Inputs

| Input | Why it matters |
|---|---|
| Market/category/date | Determines availability and exclusivity |
| Creative format | Static/video support can vary |
| Approval timeline | Managed formats need review |
| Destination | High reach exposes weak landing pages quickly |
| Follow-up | Spark/In-Feed/Search can continue the journey |

## Measurement

Use reach, frequency, CTR, video metrics if applicable, profile/site visits, search/direct lift, and downstream campaign impact.

## Common Mistakes

- Trying to communicate too much in a very short format.
- Treating quoted pricing from one market/date as universal.
- No retargeting or follow-up plan.
- Weak landing page or app store experience.
